# Microsoft FY25 Kurumlar Vergisi Raporu - Türkçe Web Sunumu

Bu klasör, FY25 Microsoft EU Directive 2021/2101 ülke bazlı kurumlar vergisi raporunun Türkçe statik web sunumunu içerir.

## Canlı adres

https://atpsec.github.io/wc26-ucunculer/microsoft-tax-turkce-web/

## İçerik

- Türkçe rapor özeti
- Türkiye özel görünümü
- Ülke bazlı gelir, kâr, vergi ve çalışan tablosu
- Arama ve sıralama
- Türkiye'deki iştirakler

## Not

Bu web sunumu hukuki, finansal, muhasebesel veya vergi danışmanlığı değildir. Rakamlar kaynak PDF raporundaki USD değerlerinden alınmıştır.
